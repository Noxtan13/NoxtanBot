Discord

Discord Developer Webseite öffnen Mit Discord Acc anmelden New Application - Neue Anwendung erstellen Name und Beschreibungen ausfüllen Links im Menü auf den Punkt Bot klicken Daten des Bots hinterlegen (so taucht dieser auf Discord auf) Gateway Intens auswählen, auf dessen Events reagiert werden soll Bot Permissions können über die Anwendung erteilt werden oder über die Webseite Links im Menü auf OAuth2 klicken Unter General kann bei Client Information die Client ID kopiert werden Diese in das Feld Client ID eintragen Die Berechtigungen des Bots auswählen (Admin ist automatisch alles) Auf Button Server joinen drücken Im Webbrowser mit Discord-Account anmelden (wenn nicht schon angemeldet ist) Im Webbrowser den Discord-Server auswählen, dem der Bot joinen soll. Dann auf Weiter Berechtigungen prüfen und evtl. richtig auswählen. Mit Authorisieren wird der Bot dem Server joinen Captcha lösen Damit Bot eingelogt wird: In der Anwendung noch auf Token holen klicken Es wird die Discord Developer Seite aufgerufen Wenn mal ein Token erstellt worden ist auf "Reset Token" klicken (vorherige Token werden dann ungültig) Wenn Token angezeigt wird auf Copy klicken Den Token in das Feld einfügen Fenster schliessen, damit Daten gespeichert werden
