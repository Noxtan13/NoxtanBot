Discord

Bot auf Discord registrieren.
  1. Discord Developer Webseite öffnen unter diesen Link.
  2. Mit Discord Account oben rechts anmelden
  3. New Application - Neue Anwendung erstellen
  4. Name und Beschreibungen ausfüllen
  5. Links im Menü auf den Punkt Bot klicken
  6. Daten des Bots hinterlegen (so taucht dieser auf Discord auf)
  7. Gateway Intens auswählen, auf dessen Events reagiert werden soll
  8. Bot Permissions können über die Anwendung erteilt werden oder über die Webseite
  9. Links im Menü auf OAuth2 klicken
  10. Unter General kann bei Client Information die Client ID kopiert werden
  11. Im NoxtanBot: Die kopierte Client ID in das Feld Client ID eintragen
  12. Die Berechtigungen des Bots auswählen (Admin ist automatisch alles)
  13. Auf Button Server joinen drücken
  14. Im Webbrowser mit Discord-Account anmelden (wenn nicht schon angemeldet ist)
  15. Im Webbrowser den Discord-Server auswählen, dem der Bot joinen soll. Dann auf Weiter
  16. Berechtigungen prüfen und evtl. richtig auswählen. Mit Authorisieren wird der Bot dem Server joinen
  17. Captcha lösen
Einloggen des Bots
  1. In der Anwendung auf Token holen klicken
  2. Es wird die Discord Developer Seite aufgerufen
  3. Wenn ein Token erstellt worden ist auf "Reset Token" klicken (vorherige Token werden dann ungültig)
  4. Wenn Token angezeigt wird auf Copy klicken
  5. Den Token in das Feld einfügen
  6. Fenster schliessen, damit Daten gespeichert werden
