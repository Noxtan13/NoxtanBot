Bot bei Twitch Registrieren
  1. Die Twitch Developer Console aufrufen unter https://dev.twitch.tv/console.
  2. Unter "Anwendungen" mit "Anwendung registrieren" einen Bot erstellen.
  3. Dazu einfach Name + Kategorie auswählen.
  4. Unter "OAuth Redirect URLs" werden die Links eingetragen, an denen Twitch die Antwort der Berechtigungen sendet.
    • Damit das Programm des Bots die Berechtigungen abfragen kann, muss sollte der Wert "https://twitchapps.com/tokengen/" eingetragen werden.
    • Sollen andere Adressen verwendet werden, können bis zu 5 eingetragen werden.
    • Wird als redirect ein Wert eingegeben, welches nicht in der Liste steht, so wird kein Token erzeugt.
  6. Wähle in der Kategorie "Chat Bot" aus.
  7. Mit "Erstellen" den Bot erstellen
Twitch-Token für den Bot anfordern
Um Twitch-Token anzufragen, muss je nach Funktionsumfang zweimal der Token angefordert werden
    • Wähle die verfügbaren Berechtigungen aus, die der Bot braucht.
    • Damit der Chat funktioniert sind die Berechtigungen "Chat_read" und "Chat_edit" mind. notwendig.
  2. Fordere mit dem Button "Berechtigungen anfordern" die Twitch-Berechtigungen für den Bot an
  3. Es öffnet sich im Browser die Twitch-Seite zum Bestätigen der angeforderten Berechtigungen
  4. Diese Bestätigen und wenn das Fenster von Tokengen anzeigt wird, den Wert kopieren
  5. Den kopierten Schlüssel in das richtige Feld einfügen.
  6. Sobald die Twitch-Einstellungen geschlossen werden erscheint die Abfrage, ob die neuen Tokens übernommen werden sollen
  7. Damit die Tokens übernommen werden, muss der Bot sich nochmal neu an Twitch anmelden (falls er aktuell gestartet ist)
