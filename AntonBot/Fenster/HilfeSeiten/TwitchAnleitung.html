Bot bei Twitch Registrieren
 1. Die Twitch Developer Console aufrufen unter https://dev.twitch.tv/console.
2. Unter "Anwendungen" mit "Anwendung registrieren" einen Bot erstellen.
3. Dazu einfach Name + Kategorie auswählen.
4. Unter "OAuth Redirect URLs" werden die Links eingetragen, an denen Twitch die Antwort der Berechtigungen sendet.
--> Damit das Programm des Bots selber die Berechtigungen abfragen kann, muss hier der Wert "https://127.0.0.1:|PortNr|
--> Beim Start der TwitchEinstellungen wird eine freie Port-Nummer gefunden und angezeigt. Diese muss an der Stelle von |PortNr| kopiert werden
--> Kann später aber noch angepasst werden.
5. Wähle in der Kategorie "Chat Bot" aus.
6. Mit "Erstellen" den Bot erstellen
Twitch-Token für den Bot anfordern Um Twitch-Token anzufragen, muss die OAuth Redirect URL angepasst werden.
1. In der Twitch Develogper Console den Bot bearbeiten mit "Verwalten"
2. Unter den OAuth Redirect URLs die Adresse https://127.0.0.1: eintragen.
3. Nach dem Doppelpunkt wird der Port eingetragen, der im Twitch-Einstellungsfenster bei Port angezeigt wird
--> Dieser Port mit jedem Aufruf des Fensters neu generiert und ist ein aktuell freier Prot am PC
4. Drücke auf Speichern, damit die OAuth Redirect URL gespeichert wird.
5. Kopiere aus der Twitch-Developer Konsole die Client-ID und füge diese in das Textfeld im Twitch-Einstellungsfenster ein
6. Wähle die verfügbaren Berechtigungen aus, die der Bot braucht.
--> Damit der Chat funktioniert sind die Berechtigungen "Chat_read" und "Chat_edit" mind. notwendig.
7. Fordere mit dem Button "Berechtigungen anfordern" die Twitch-Berechtigungen für den Bot an
8. Es öffnet sich im Browser die Twitch-Seite zum Bestätigen der angeforderten Berechtigungen
9. Diese Bestätigen und wenn das Fenster angezeigt wird zum schliessen des Browsers, sind die Berechtigungen gespeichert
10. Sobald die Twitch-Einstellungen geschlossen werden erscheint die Abfrage, ob die neuen Tokens übernommen werden sollen
11. Damit die Tokens übernommen werden, muss der Bot sich nochmal neu an Twitch anmelden (falls er aktuell gestartet ist)
